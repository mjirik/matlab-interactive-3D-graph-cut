matlab-interactive-3D-graph-cut
===============================

Interactive 3D Graph Cut for Matlab


Install
-------

Get submodules

    git submodule init

    git submodule update

Run install.m

    matlab -nodesktop -nosplash -r "install; quit"

Try examples

* example00.m - grayscale 2D
* example10.m - grayscale 3D
* example12.m - color 2D
